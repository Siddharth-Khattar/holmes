<task type="auto">
  <name>Task 2: Configure design tokens in globals.css</name>
  <files>frontend/src/app/globals.css</files>
  <action>
    Replace the current globals.css with a complete design token setup using Tailwind v4's @theme directive.

    Include:
    - Color palette from CONTEXT.md:
      - charcoal: #121218 (deepest darks, background base)
      - jet: #22333B (cool teal-black, secondary accent)
      - smoke: #F2F4F3 (text, highlights)
      - taupe: #A9927D (primary warm accent)
      - stone: #5E503F (supporting warm tone)
    - Glass effect colors:
      - glass-light: rgba(242, 244, 243, 0.08)
      - glass-dark: rgba(18, 18, 24, 0.6)
    - Custom shadows for glass depth:
      - shadow-glass: 0 8px 32px rgba(0, 0, 0, 0.12)
      - shadow-glass-hover: 0 12px 40px rgba(0, 0, 0, 0.16)
      - shadow-glow-taupe: 0 0 40px rgba(169, 146, 125, 0.15)
    - Font reference: --font-serif: var(--font-fraunces)

    Add @layer utilities for Fraunces variable font axes:
    - .font-wonky (WONK: 1)
    - .font-soft (SOFT: 100)
    - .font-sharp (SOFT: 0)
    - .font-editorial (WONK: 1, SOFT: 50, optical sizing auto)

    Add @layer components for glass utilities:
    - .glass-panel (light glass for cards)
    - .glass-nav (darker glass for navigation)

    Add reduced motion media query for accessibility.

    Set body base styles to use charcoal background and smoke text.
  </action>
