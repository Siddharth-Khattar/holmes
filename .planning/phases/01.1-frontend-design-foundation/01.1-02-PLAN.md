<task type="auto">
  <name>Task 1: Create animation utilities library</name>
  <files>frontend/src/lib/animations.ts</files>
  <action>
    Create frontend/src/lib/animations.ts with shared animation variants.

    Include these Motion variants:

    1. `fadeInUp` - Standard reveal animation:
       - hidden: { opacity: 0, y: 20 }
       - visible: { opacity: 1, y: 0, transition: { duration: 0.6, ease: "easeOut" } }

    2. `fadeIn` - Simple fade without movement:
       - hidden: { opacity: 0 }
       - visible: { opacity: 1, transition: { duration: 0.5 } }

    3. `staggerContainer` - Parent variant for staggered children:
       - hidden: { opacity: 0 }
       - visible: { opacity: 1, transition: { staggerChildren: 0.15, delayChildren: 0.1 } }

    4. `scaleOnHover` - Hover interaction for cards:
       - { scale: 1.02, transition: { duration: 0.2 } }

    5. `slideInFromLeft` - For alternating layout sections:
       - hidden: { opacity: 0, x: -30 }
       - visible: { opacity: 1, x: 0, transition: { duration: 0.5, ease: "easeOut" } }

    6. `slideInFromRight` - Counterpart:
       - hidden: { opacity: 0, x: 30 }
       - visible: { opacity: 1, x: 0, transition: { duration: 0.5, ease: "easeOut" } }

    Export all variants with proper TypeScript types (use Motion's Variants type).

    Add ABOUTME comment at top explaining this is shared animation configuration.
  </action>
  <verify>
    - File compiles without TypeScript errors
    - All variants are exported
    - Types are correctly defined
  </verify>
  <done>Animation variants library created with all standard animation patterns.</done>
</task>
